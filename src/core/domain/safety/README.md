## üöÄ Quick Start

### Basic Usage

```python
from core.domain.safety import AdvancedContentFilter, SafetyConfig

# Initialize with default configuration
safety_filter = AdvancedContentFilter()

# Analyze content

    text="Let's learn about colors! What's your favorite color?",
    child_age=5
)
```
